1. 개요

AI모아(이하 "서비스")는 이용자의 개인정보를 중요하게 생각합니다. 본 방침은 서비스 이용 과정에서 어떤 정보가 수집되고 어떻게 사용되는지 설명합니다.

2. 수집하는 정보

AI모아는 회원가입, 로그인 등의 기능을 제공하지 않으며, 별도의 개인정보를 수집하지 않습니다.

다만 아래와 같은 기술적 정보가 자동으로 처리될 수 있습니다.

항목 목적 보존 기간
로컬스토리지 (다크모드 설정) 사용자 화면 설정 유지 브라우저 삭제 시까지
이메일 (문의 시 직접 제공) 문의 답변 답변 완료 후 즉시 삭제

3. 쿠키 및 로컬스토리지

서비스는 쿠키를 사용하지 않습니다. 다크모드 설정 등 UI 상태는 브라우저의 로컬스토리지에 저장되며, 서버로 전송되지 않습니다. 브라우저 설정에서 언제든지 삭제할 수 있습니다.

4. 제3자 서비스

서비스는 아래 제3자 서비스를 이용하며, 각 서비스의 개인정보 처리방침이 적용됩니다.

  • Google Fonts (폰트 로드)
  • Cloudflare CDN (Font Awesome, Tailwind CSS)
  • GitHub Pages (서비스 호스팅)
  • Google Material Symbols (아이콘)

5. 외부 링크

서비스 내 AI 툴 링크를 통해 이동하는 외부 사이트에서의 개인정보 처리는 해당 사이트의 정책을 따릅니다. AI모아는 이에 대한 책임을 지지 않습니다.

6. 정보주체의 권리

서비스는 별도의 개인정보를 수집하지 않으므로 열람, 수정, 삭제 요청 대상 데이터가 존재하지 않습니다. 이메일 문의를 통해 제공하신 정보의 삭제를 원하시면 아래로 연락 주세요.
